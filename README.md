# Claude Code Plugins Marketplace

Claude Code の機能を拡張するプラグインのマーケットプレイスです。

## 利用方法

```shell
# マーケットプレイスの追加
/plugin marketplace add canpok1/claude-code-plugins

# プラグインのインストール
/plugin install github@canpok1-plugins
```

## プラグイン一覧

| プラグイン名 | 説明 |
|-------------|------|
| [github](plugins/github/) | GitHub 連携プラグイン（開発中） |
